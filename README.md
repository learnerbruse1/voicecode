# VoiceCode 🎙️

本地语音转文字工具，专为 vibe coding 设计。完全离线，支持中英文，低资源占用。

## 特性

- 🔒 **完全本地** — 无需联网，数据不上云
- 🌐 **中英双语** — 自动检测语言
- ⚡ **低占用** — 默认 base 模型，仅需 ~150MB 内存
- ⌨️ **快捷键** — 空格键一键开始/停止录音
- 🌙 **深色UI** — 浏览器界面，无需安装客户端

## 系统要求

- Windows 10/11
- Python 3.10+
- 麦克风

## 快速开始

```bat
# 1. 安装（仅首次）
setup.bat

# 2. 启动
run.bat
```

浏览器会自动打开 `http://localhost:7788`

## 模型对比

| 模型 | 大小 | 速度 | 精度 |
|------|------|------|------|
| tiny | ~75MB | 最快 | 一般 |
| **base** | ~150MB | **快（推荐）** | **好** |
| small | ~500MB | 中等 | 很好 |
| medium | ~1.5GB | 慢 | 极好 |

切换模型：设置环境变量 `WHISPER_MODEL=small` 后重启，或在UI的下拉菜单选择（需重启后生效）。

## 使用方法

1. 启动后浏览器自动打开
2. 按 `空格键` 开始录音（按钮变红）
3. 再按 `空格键` 停止，稍后显示转录结果
4. 点击 **Copy** 复制文字

## 手动安装

```bash
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
.venv\Scripts\python app.py
```

## 常见问题

**Q: 麦克风无法访问？**  
A: 浏览器会弹出权限请求，点击"允许"；或检查 Windows 隐私设置 → 麦克风。

**Q: 首次启动很慢？**  
A: 正在自动下载 Whisper 模型，下载完成后会快很多。

**Q: 如何提高准确率？**  
A: 在 UI 中切换到 `small` 或 `medium` 模型，重启服务。
